# Data

The raw dataset is not included in this repository.

## Source

**IBM Telco Customer Churn Dataset**
Available on Kaggle: https://www.kaggle.com/datasets/blastchar/telco-customer-churn

## Dataset Details

- 7,043 customer records
- 33 features including demographics, services, contract type, and payment method
- Target variable: Churn Label (Yes/No)
- Geography: California, United States

## Instructions

1. Download `WA_Fn-UseC_-Telco-Customer-Churn.csv` from the Kaggle link above
2. Place it in this `data/` folder
3. The notebook will load it from this path automatically
